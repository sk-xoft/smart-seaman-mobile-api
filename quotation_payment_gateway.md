## ขอบเขตงาน (Scope of Work)

- พัฒนา Payment Gateway Integration สำหรับ Smart Seaman Mobile API (Backend)
- รองรับ 3 ช่องทาง: **PromptPay**, **Credit/Debit Card**, **Internet Banking**
- ภาษา/Framework: Java 11, Spring Boot 2.6.x
- รวม Unit Test และ Integration Test บน Sandbox environment

## ไม่รวมในขอบเขตงาน

- งาน Frontend / Mobile UI
- ค่าธรรมเนียม Payment Provider (Omise, GB Prime Pay, 2C2P ฯลฯ)
- ค่า Infrastructure / Server
